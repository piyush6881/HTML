1.How many headings are there?

There are 6 levels of headings in HTML from h1 to h6, where h1 is the highest most important and h6 is the lowest or least important can be used multiple times.

2.H1 can be used how many times?

h1 tag can be used many times as needed within an html document, however it is recommended to be used only once per page to represent main heading of the document, for better SEC and accessibility practice all subsequent headings to be from h2 to h6.

3. What is self-closing tag, empty tag, and void tag?

Self-Closing, empty or Void Tag: A Tag that does not have a closing tag, they do not have any content. Images and links cannot have content - they are pointers to an element installed on the website example are break br tag, img or image tag, or hr, link etc.

4.Why do we use < br > tag?

br tag is used to insert line break in the text offen used for text formatting to start new line without starting a new paragraph.

5.Why do we use < hr > tag?

hr tag is used to create a horizontal rule or line across the page. It is used to visually separate sections of content.

6.What is <marquee> tag?

The marquee tag creates a scrolling text or image on a webpage. Now obsolete or is not recommended, presently CSS animations and JavaScript are instead used for creating scrolling effect.

7.Difference between block-level and inline element?

Block level Element takes full width available 100% of the parent container, like div, h1, p, etc. where else, inline element takes only as much width as necessary it does not start on a new line. Example lebel, span etc.

8.Div and span tag difference?

Div tag is block level element and span tag is inline element. Say you want to highlight one word in the text in line and you want to highlight the entire text so to highlight only one work inline element is used.
