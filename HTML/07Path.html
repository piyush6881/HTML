Types of Paths in HTML

Summary
Absolute Path:Provides the complete URL, useful for external links.
Relative Path:Provides the location relative to the current file, useful for internal links within the same website.
  1. Absolute Path:
  • An absolute path specifies the complete URL to a file.
  • It starts from the root directory and includes the full URL.
  • Useful for linking to resources located on different websites.
  • or in our case some folder while creating the website
  1. Relative Path:
  • A relative path specifies the location of a file relative to the current document.
  • Does not include the full URL, just the path from the current location.
  • Useful for linking to resources within the same website or directory structure.
  • Or in our case sub-folder with ../ (2 dots and space)
  • This gives you drop down of folders then file or image path
