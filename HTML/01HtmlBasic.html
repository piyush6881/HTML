HtmlBasic

"!DOCTYPE html"

Meaning:This defines the document type and version of HTML being used.

In this case, it specifies HTML5.

Use:It ensures that the browser interprets the document as an HTML5 document.

"html lang="en""

Meaning:This is the root element of an HTML document.

"lang":Attribute specifies language of the document. "en" stands for English.

This helps search engines and browsers understand the primary language of the content.

"head"

Meaning:The head element contains meta-information only for browser to understand about document.

Use:Include metadata (like character set, page title, and viewport settings) that does not get displayed directly on the web page.

"meta name="viewport" content="width=device-width, initial-scale=1.0"

Meaning:This meta tag provides information about how the web page should be displayed on different devices.

Use:Width = device-width sets the width of the page or viewport to the width of the device,

and initial-scale=1.0 sets the initial zoom level when the page is first loaded

This helps ensure that the page is responsive and displays well on both desktop and mobile devices.
