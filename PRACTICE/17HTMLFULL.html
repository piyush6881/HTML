BASIC HTML

<h>HEADER Text From h1 to h6 </h>

<p> Defines a paragraph.paragraph or you can also use just to give break Line Break Tag <br> the break

Makes the text bold. <b> Italicizes the text. <i>

<div>Defines a division or section in an HTML document.
Often used as a container for other HTML elements to style them with CSS or manipulate them with JavaScript.

<img> Attributes:
src: Specifies the path to the image.
alt: Provides alternative text if the image cannot be displayed.
width: Specifies the width of the image.
height: Specifies the height of the image.


Abbreviation Tag <abbr>: Defines an abbreviation or acronym.
Can provide a full description when hovered over in some browsers.
Attribute: title: Provides the full term for the abbreviation.

Table Tag

<table>: Defines a table. Common attributes: border: Specifies the width of the table border. cellpadding: Specifies the space between the cell wall and the cell content. cellspacing: Specifies the space between cells. Table Elements:
<tr>: Defines a row in a table.
<td>: Defines a cell in a table.
<th>: Defines a header cell in a table.
<thead>: Groups the header content in a table.
<tbody>: Groups the body content in a table.
<tfoot>: Groups the footer content in a table.


Form Tag

<form>: Creates an HTML form for user input.
Attributes:
action: Specifies where to send the form-data when the form is submitted.
method: Specifies the HTTP method to use when sending form-data (get or post).
enctype: Specifies how the form-data should be encoded when submitted (e.g., multipart/form-data for file uploads).
Form Elements:
<input>: Defines an input control.
<button>: Defines a clickable button.
<select>: Defines a drop-down list.
<option>: Defines an option in a drop-down list.
<label>: Defines a label for an <input> element.


Inline Frame Tag

<iframe>:
Embeds another HTML page within the current page.
Attributes:
src: Specifies the URL of the page to embed.
width: Specifies the width of the iframe.
height: Specifies the height of the iframe.
frameborder: Specifies whether or not to display a border around the iframe.
allow: Specifies permissions for the iframe content (e.g., autoplay, fullscreen).
